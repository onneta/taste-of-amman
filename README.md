# Taste of Amman

restaurant application generated by ONI.

## Features
- Online ordering system with payment processing and delivery integration
- Customer loyalty program with segmented email marketing
- Real-time revenue and food cost tracking dashboard
- Google Business Profile optimization and review generation
- Instagram Reels content calendar and social media engine

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
